# Kerala Road Accident Analysis

## Project Overview

This project performs a complete Exploratory Data Analysis (EDA) on a Kerala Road Accident dataset.

The goal is to identify accident patterns, severity factors, temporal trends, and safety-related insights using statistical and visualization techniques.

---

## Dataset Information

* Rows: ~5,700+
* Columns: 20+
* Numerical Features
* Categorical Features
* Date and Time Features
